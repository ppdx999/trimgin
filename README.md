# trimgin (trim + margin)
Small Command to remove leading and trailing whitespace from text.

# Usage

```terminal
$ cat text.txt


Hi there!
We have two blank lines above and below this text.
And this space will be removed by 'trimgin'.


```

```terminal
$ trimgin text.txt
Hi there!
We have two blank lines above and below this text.
And this space will be removed by 'trimgin'.
```
